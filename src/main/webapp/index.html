1. 顺序访问P2P图书馆 API:

step1:
  • 列出第一页图书(每页10项)
  • 列出第二页图书(每页2项)
step2:
  • 登录为User1(注意保存Token)
  • User1请求借阅User2的书 (手工替换URL中的Token为前一步保存的值)
  • 列出User2借阅的图书(替换token)
step3:
  • 登录为User1
  • User2 确认图书借出
  • 列出User2拥有的图书
step4:
  • User2 确认图书归还，其他人可以借阅本书
  • 列出此书当前状态
other flow:
  • User1 cancel the apply, replace the token
  • User2 reject the apply, replace the token
